# Portfolio do Herói - Nicole Jovita

Este é um projeto de portfólio feito como prática de **HTML, CSS e JavaScript** durante a semana de capacitação da Orc'estra Gamificação, Empresa Júnior de Engenharia de Software na Universidade de Brasília.  

O projeto inclui:  
- Painel do Herói dinâmico, onde o usuário vê nome, classe, XP, nível e título.  
- Sistema de evolução de XP com alertas quando o herói sobe de nível ou muda de título.  
- Simulação de batalhas para ganhar XP (incremento de +10 XP).  
- Formulário de contato com validação de campos e mensagem de sucesso/erro sem recarregar a página.  
- Consulta a um “Oráculo” via API para mostrar mensagens aleatórias.  

## Tecnologias
- HTML5  
- CSS3  
- JavaScript (ES6)
